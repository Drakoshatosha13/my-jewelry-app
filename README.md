# Ювелирная программа 💍

Полноценное приложение для управления ювелирным магазином.

**Возможности:**
- 👑 Администратор: добавление изделий, аналитика, подробные отчеты а так же выгрузка в Excel и т.д
- 🛒 Продавец: продает изделия, может в отчетах посмотреть сколько он продал изделий и на какую сумму и т.д
